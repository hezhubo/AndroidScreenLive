# AndroidScreenLive
安卓屏幕直播示例
- [x] 支持屏幕录制保存本地
- [x] 支持RTMP屏幕直播
- [x] 支持边直播边录制保存本地
- [x] 支持视频帧率控制
- [x] 支持Android10以上混合录制麦克风及系统输出声音
- [x] 支持动态修改视频码率
- [ ] 支持视频滤镜
- [ ] 支持音频滤镜
